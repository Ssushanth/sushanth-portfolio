# Sushanth S — Portfolio (with vCard)

This portfolio highlights the professional work of **Sushanth S**, Full Stack Developer.

### Features
- Auto‑switch light/dark theme
- Sections: About, Skills, Projects, Experience, Education, Contact
- Downloadable resume and vCard
- Responsive design

### Run Locally
1. Download and unzip this folder.
2. Add your `resume.pdf` to the same folder.
3. Open `index.html` in your browser.

### Deploy
- [Vercel](https://vercel.com/)
- [GitHub Pages](https://pages.github.com/)
